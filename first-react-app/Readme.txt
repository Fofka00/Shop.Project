Запук backend (Express + API)
В папке корня проекта:

Установи зависимости:
npm install

запуск:
nodemon start.ts
Сервер должен стартовать на http://localhost:3000.



Запуск админку (Shop.Admin)
В папке
Shop.Admin:

Установи зависимости:
npm install

запуск:
node app.js
Админка будет доступна на http://localhost:4000/admin.


Запуск React‑клиента (Shop.Client)
В папке Shop.Client:

Установи зависимости:
npm install

запуск:
npm start
Сервер должен стартовать на http://localhost:3001.
